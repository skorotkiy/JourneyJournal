# JourneyJournal

A journey tracking application with a .NET Core API backend, React frontend, and Entity Framework Core data layer.

## Default Branch

This repository uses `main` as the default branch.

## Project Structure

See [docs/appstructure-create.md](docs/appstructure-create.md) for detailed information about the application structure and setup instructions.

## Architecture

- **Frontend**: React + TypeScript + Vite (JourneyJournal.Web)
- **Backend API**: .NET Core Web API (JourneyJournal.Api)
- **Data Layer**: .NET Core Class Library with Entity Framework Core (JourneyJournal.Data)

## Getting Started

Refer to the documentation in the `docs` folder for setup instructions.
